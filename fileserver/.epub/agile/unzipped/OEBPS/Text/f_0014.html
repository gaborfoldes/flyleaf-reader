1.1 Installing on Windows

The easiest way to install Rails on Windows is by using the the RailsInstaller[7] package. Just be sure that you are using version 2.0, which includes Ruby 1.9.2 and Rails 3.1. At the time of this writing, RailsInstaller 2.0 is only available as a Preview Release and appears at the bottom of the page. Feel free to use the Preview Release until the Final Release is available.

Base installation is a snap. After you download, click Run, then click Next. Select “I accept the License” (after reading it carefully of course), click Next, Install, and Finish.

This opens a command window and prompts you for your name and email. This is only to set up the git version control system. For the purposes of the exercises in this book, you won’t need to worry about the ssh key that is generated.

Close this window and open a new command prompt by selecting the Windows Start, Run..., enter cmd, and click OK.

OK, you Windows users are done. You can skip forward to Section 1.4, ​Choosing a Rails Version​. See you there.
